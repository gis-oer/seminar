# QGIS入門講習会
　QGISは、無償で利用できるオープンソースのGISとして、研究を含む幅広い分野での活用されています。この講習会では、QGISを用いて、GISの基本やソフトウェアの操作法を実習形式で解説します。今年度は異なるテーマで、2～3回のQGIS入門講習会を予定しています。※講習の前半は各回共通です。

## 第1回地形環境の分析に使えるQGIS入門
